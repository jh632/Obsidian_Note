# 中值滤波

## 概述
取滑动窗口内样本的中值，对脉冲/椒盐噪声极其有效，能保边缘。

## 公式

$$y_k = \text{median}\{x_{k-N+1},\ \dots,\ x_k\}$$

对窗口内 $N$ 个样本排序后取第 $\lfloor N/2 \rfloor$ 个值。

## 快速实现（冒泡排序取中值）
```c
typedef struct {
    float *buf;
    int    len;
    int    idx;
} median_filter_t;

static inline void median_init(median_filter_t *f, float *buf, int len) {
    f->buf = buf;
    f->len = len;
    f->idx = 0;
    // len 取奇数，如 3、5、7
}

static float median_update(median_filter_t *f, float x) {
    f->buf[f->idx] = x;
    f->idx = (f->idx + 1) % f->len;

    float tmp[8]; // 栈上拷贝，len <= 8
    memcpy(tmp, f->buf, f->len * sizeof(float));

    // 冒泡排序
    for (int i = 0; i < f->len - 1; i++)
        for (int j = 0; j < f->len - 1 - i; j++)
            if (tmp[j] > tmp[j + 1]) {
                float t = tmp[j]; tmp[j] = tmp[j + 1]; tmp[j + 1] = t;
            }

    return tmp[f->len / 2];  // 取中值
}
```

## 3 元素特化版（零排序）
```c
static inline float median3(float a, float b, float c) {
    if (a > b) { float t = a; a = b; b = t; }
    if (b > c) { float t = b; b = c; c = t; }
    if (a > b) b = a;
    return b;
}
```

## 要点
- 窗口一般取 3/5/7，再大排序开销显著
- 3 元素时用特化版，仅 3 次比较，无排序
- 中值滤波不改变信号幅度，天然保边缘
