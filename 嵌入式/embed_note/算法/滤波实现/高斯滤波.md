# 高斯滤波

## 概述
窗口内样本按高斯核加权平均，近处权重大远处权重小，比均值滤波更平滑且边缘保持更好。

## 公式

1D 高斯核函数：
$$g(i) = \frac{1}{\sigma\sqrt{2\pi}}\,\exp\!\left(-\frac{i^2}{2\sigma^2}\right)$$

卷积输出：
$$y_k = \sum_{i=-M}^{M} g(i)\cdot x_{k-i}, \quad M = \lfloor N/2 \rfloor$$

归一化：$\sum g(i) = 1$。

## 1D 高斯卷积实现
```c
#include <math.h>

// 生成高斯核（归一化），sigma 取 0.8~1.5 常用
static void gauss_kernel(float *k, int len, float sigma) {
    float sum = 0;
    int half = len / 2;
    for (int i = 0; i < len; i++) {
        float d = i - half;
        k[i] = expf(-(d * d) / (2 * sigma * sigma));
        sum += k[i];
    }
    for (int i = 0; i < len; i++) k[i] /= sum;
}

typedef struct {
    float *buf;
    float *kern;
    int    len;
    int    idx;
} gauss_filter_t;

static inline void gauss_init(gauss_filter_t *f, float *buf, float *kern, int len) {
    f->buf  = buf;
    f->kern = kern;
    f->len  = len;
    f->idx  = 0;
    memset(buf, 0, len * sizeof(float));
}

static float gauss_update(gauss_filter_t *f, float x) {
    f->buf[f->idx] = x;
    f->idx = (f->idx + 1) % f->len;

    float out = 0;
    for (int i = 0; i < f->len; i++) {
        int bi = (f->idx + i) % f->len;
        out += f->buf[bi] * f->kern[i];
    }
    return out;
}
```

## 整数近似核（5 点，省 FPU）
```c
// 核: [1, 4, 6, 4, 1]，除以 16 = 右移 4
// sigma ≈ 1.0
static int16_t gauss5_int(int16_t *buf) {
    return (buf[0] + 4 * buf[1] + 6 * buf[2] + 4 * buf[3] + buf[4]) >> 4;
}
```

## 要点
- sigma 越大平滑越强，窗口长度通常取 6*sigma+1
- 嵌入式可用整数近似核避免浮点运算
- 可分离性：2D 高斯 = 水平 1D + 垂直 1D，复杂度从 O(N²) 降到 O(2N)
