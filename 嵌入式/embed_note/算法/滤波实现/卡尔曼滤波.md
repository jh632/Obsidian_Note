# 卡尔曼滤波

## 概述
基于状态空间模型的最优递推估计器，融合预测（模型）与观测（传感器），在 IMU/GPS/机器人中效果最优。

## 公式（1D 静态模型）

**预测：**
$$\hat{x}_{k|k-1} = \hat{x}_{k-1}, \quad P_{k|k-1} = P_{k-1} + Q$$

**更新：**
$$K_k = \frac{P_{k|k-1}}{P_{k|k-1} + R}$$

$$\hat{x}_k = \hat{x}_{k|k-1} + K_k(z_k - \hat{x}_{k|k-1})$$

$$P_k = (1 - K_k)\,P_{k|k-1}$$

其中 $z_k$ 为观测值，$K_k$ 为卡尔曼增益。

## 标量（1D）卡尔曼实现
```c
typedef struct {
    float x;   // 状态估计
    float P;   // 估计协方差
    float Q;   // 过程噪声
    float R;   // 测量噪声
} kalman1d_t;

static inline void kalman1d_init(kalman1d_t *k, float Q, float R) {
    k->x = 0;
    k->P = 1;  // 初始不确定性
    k->Q = Q;
    k->R = R;
}

static inline float kalman1d_update(kalman1d_t *k, float z) {
    // 预测（1D 静态模型: x(k+1) = x(k)）
    k->P += k->Q;

    // 更新
    float K = k->P / (k->P + k->R);  // 卡尔曼增益
    k->x += K * (z - k->x);
    k->P *= (1 - K);

    return k->x;
}
```

## 调参
| 参数 | 含义 | 增大效果 |
|------|------|----------|
| Q | 过程噪声 | 更信任观测，响应更快但更抖 |
| R | 测量噪声 | 更信任模型，输出更平滑但滞后 |

经验：先设 Q=0.001、R=1.0，再根据实际响应调整。

## 要点
- 1D 静态模型假设状态不变，适合温度/电压等缓变量
- 有运动模型时用矩阵形式（位置+速度），适合 IMU/定高
- 每次更新仅需几次乘加，嵌入式完全可行
