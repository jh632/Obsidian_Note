# 均值滤波

## 概述
对滑动窗口内样本取算术平均，抑制高斯白噪声，实现极简，嵌入式最常用。

## 公式

直接形式：
$$\bar{x}_k = \frac{1}{N}\sum_{i=0}^{N-1} x_{k-i}$$

递推形式（O(1)）：
$$\bar{x}_k = \bar{x}_{k-1} + \frac{x_k - x_{k-N}}{N}$$

## 递推实现（O(1) 更新）
```c
#include <string.h>

typedef struct {
    float *buf;
    int   len;
    int   idx;
    float sum;
} mean_filter_t;

static inline void mean_init(mean_filter_t *f, float *buf, int len) {
    f->buf = buf;
    f->len = len;
    f->idx = 0;
    f->sum = 0;
    memset(buf, 0, len * sizeof(float));
}

static inline float mean_update(mean_filter_t *f, float x) {
    f->sum += x - f->buf[f->idx];   // 减最老，加最新
    f->buf[f->idx] = x;
    f->idx = (f->idx + 1) % f->len;
    return f->sum / f->len;
}
```

## 要点
- 递推形式每次只需 1 次加 + 1 次减 + 1 次除，无需遍历窗口
- 窗口越大平滑越强，但延迟也越大（延迟 = N/2 个采样周期）
- 对脉冲噪声无抵抗力，尖峰会拉偏整个窗口
