
### 1 测试方法

测试环境:室内轻度干扰环境
测试端流程

| 步骤                | 作用                   |
| ----------------- | -------------------- |
| 初始化 NVS           | 蓝牙协议栈需要用到 NVS        |
| 释放 Classic BT 内存  | 只用 BLE，不用传统蓝牙，节省 RAM |
| 初始化 BT Controller | 初始化底层蓝牙控制器           |
| 启用 BLE 模式         | 只开启 BLE              |
| 初始化并启用 Bluedroid  | 启动 ESP-IDF 默认蓝牙协议栈   |
| 注册 GAP 回调并设置扫描参数  | 开始 BLE 扫描流程          |

## 3 测试总结

测试结果表明，外接天线对 BLE 信号质量有明显改善。在 0 m 距离下，RSSI 从 -70 dBm 提升至 -47 dBm，提升约 23 dB；在 5 m 距离下，RSSI 从 -100 dBm 提升至 -82 dBm，提升约 18 dB。未连接天线时，10 m 距离下串口无扫描日志，无法检测到有效 BLE 信号；连接天线后，10 m 距离下仍可检测到信号，RSSI 约为 -99 dBm。

因此，在当前室内轻度干扰环境下，连接天线后 BLE 接收信号强度提升约 18–23 dB，最大有效测试距离由约 5 m 提升至约 10 m，距离提升约 1 倍，说明天线对 BLE 通信可靠性和覆盖范围有显著改善。
